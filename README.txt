APPSUITE — ALL-IN-ONE (NO TERMINAL)

Ini 1 aplikasi Windows (Desktop HTA) yang menggabungkan semua fitur:
1) GoFocusFlow (Focus sessions)
2) PomodoroPulse (Pomodoro + sessions)
3) BudgetBuddy (Budget tracker)
4) FileScout (File search)
5) PasswordForge (Password generator)
6) FocusBoard (To Do / Doing / Done board)
7) DeskMini (Quick tools + sticky notes)
8) CSV QuickView (CSV viewer)
9) StopwatchX (Stopwatch + laps)

Cara menjalankan:
1) Extract ZIP (jangan dijalankan dari dalam zip)
2) Double-click: Run-AppSuite.vbs

Jika SmartScreen muncul:
- More info -> Run anyway
- atau: klik kanan ZIP/folder -> Properties -> Unblock -> Apply

Data disimpan otomatis di folder:
data\*.json

Catatan:
Aplikasi memakai mshta.exe (bawaan Windows). Jika mshta diblokir oleh kantor/sekolah, aplikasi tidak bisa dibuka.
